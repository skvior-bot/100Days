# 交互管理

> 职责：整理所有可交互项的信息（名称、跳转项、交互树结构）
> 参考依据：设计参考.md > 交互设计

---

## 主界面交互

| 交互项 | 功能 | 跳转/备注 |
|-------|------|----------|
| 新的开始 | 进入游戏，开始新的轮回 | → 初始剧情 |
| 继续游戏 | 载入玩家存档 | → 弹出存档列表 → 选择 → 载入 |
| 设置 | 画面、文字效果、音量等 | → 设置界面（可能包含操作提示） |
| 退出游戏 | 关闭程序 | |

---

## 室内交互（家中）

| 交互项 | 触发方式 | 功能选项 | 跳转/备注 |
|-------|---------|---------|----------|
| 电脑 | 点击 | 游戏开发、娱乐、工作、网购 | 详见"电脑交互树" |
| 手机 | 点击 | 聊天、存档、读档、网购、发布视频、看漫画/小说、点外卖 | 随身使用 |
| 床 | 点击 | 睡觉恢复体力、度过时间 | |
| 书架 | 点击 | 阅读书籍 | |
| 门（家） | 点击 | 前往其他地点 | 详见"地点切换" |

### 电脑交互树
1. **游戏开发** → 程序/美术/音乐/文案（2×2排列）→ 选4次 → 放入结算器 → 结算
2. **娱乐** → 看动画 / 玩游戏 / 看课程 → 选择具体项目 → 放入结算器 → 结算
3. **工作** → 直播（直播游戏/杂谈）/ 发视频（已通关游戏/已看完动画）→ 结算
4. **网购** → 日常类 / 娱乐类 / 设备类 → 选择多个 → 结算

## 室外交互

| 交互项 | 地点 | 功能选项 | 跳转/备注 |
|-------|------|---------|----------|
| 收银员 | 书店 | 买书、回收书籍 | 买书按类别（漫画/小说/资料）分类 |
| 服务员 | 餐馆 | 吃饭、打工 | 受时间段限制 |
| 公园长椅 | 公园 | 散步 | |
| 收银台 | 便利店 | 购买食物和道具 | |
| 店员 | 专业道具店 | 购买专业设备 | |

### 地点切换

点击门（任何建筑物内）→ 弹出可前往地点列表（不包括当前所在地）→ 选择 → 结算 → 到达

| 出发地 | 可前往 | 消耗时间 | 消耗体力 |
|-------|--------|---------|---------|
| 家 | 书店 | 10min | 2 |
| 家 | 餐馆 | 待定 | 待定 |
| 家 | 便利店 | 待定 | 待定 |
| 家 | 公园 | 待定 | 待定 |
| 家 | 专业道具店 | 15min | 3 |
| 书店 | 家 | 10min | 2 |
| 书店 | 专业道具店 | 5min | 1 |
| 书店 | ... | 待定 | 待定 |

> 每个外出地点都可能触发角色剧情，能触发时会有特定标识

---

## 触发位置说明

- 电脑：在家中
- 手机：随身（任何时候可用）
- 床：在家中（卧室）
- 书架：在家中
- 门（家→外出）：家中
- 门（外出→家/其他）：各室外场所
- 收银员/服务员/店员：对应的室外场所

---

## 时间段限制

| 时间段 | 时间 | 书店 | 餐馆（打工） | 餐馆（吃饭） |
|-------|------|------|-------------|-------------|
| 上午 | 8:00-12:00 | ✅ 开放 | ✅ 可打工 | ❌ 不可吃饭 |
| 中午 | 12:00-14:00 | ❌ 关闭 | ❌ 不可打工 | ✅ 可吃饭 |
| 下午 | 14:00-18:00 | ✅ 开放 | ✅ 可打工 | ❌ 不可吃饭 |
| 傍晚 | 18:00-20:00 | ❌ 关闭 | ❌ 不可打工 | ✅ 可吃饭 |
| 夜晚 | 20:00-24:00 | — | — | — |
| 深夜 | 24:00-4:00 | — | — | — |

> 时间段以开头时间为主（8:00算上午，12:00算中午）
> 没写出来的默认每个时间段都可以做
