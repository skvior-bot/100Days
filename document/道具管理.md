## 购买方式

| 方式 | 消耗时间 | 等待时间 | 说明 |
|------|:--------:|:--------:|------|
| 网购 | 5min/次 | 一天（今天买明天到） | 电子版无需等待 |
| 书店购买 | 5min/次 | 即时 | |
| 便利店购买 | 即时 | 即时 | |
| 专业道具店 | 即时 | 即时 | |
| 外卖 | 待定 | 即时 | 到家或直接吃 |

---

## 备注

- 体力获取量与消耗的时间应成正比
- 游戏主要是电子版（网购时无需等待时间）
- 小说、漫画主要是实体版（网购时需要等待一天），电子版不需要钱（在手机上看的）
