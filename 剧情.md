# 剧情大纲

> 职责：整理游戏的故事情节、角色个人剧情、支线剧情

---

## 总体结构

核心：SLG（有限操作下做出最优选择）+ Gal（以剧情驱动核心玩法）
根据游戏完成度解锁不同结局 → 真结局后可玩到制作完成的游戏

---

## 游戏全流程

```
.exe启动 → 主界面（标题页）
  → 新的开始 → 初始剧情（背景+目标，可能漫画形式）→ 教程（鼠标交互）
  → 正式游戏（目标：把开发的游戏做完）
  → 结局判定
  → [真结局] → endPV（CG+名单）→ 标题/背景改变（100Days → AloneLife）
  → 进入内层游戏（玩家开发完成的游戏）
  → 通关后双标题切换 → 二周目（可能无新内容）
```
