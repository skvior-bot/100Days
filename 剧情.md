# 剧情大纲与分支

> 最后更新：2026-06-06

---

## 总体结构

- 核心：**SLG（有限操作下做出最优选择） + Gal（剧情驱动核心玩法）**
- 根据游戏完成度解锁不同结局
- 制作的游戏是固定的（"一个人的孤独之旅"），随剧情解锁不同游戏阶段
- 真结局到达后可玩到制作完成的游戏

---

## 游戏全流程

```
.exe启动 → 主界面（标题页）→ 新的开始 → 初始剧情（交代背景+最终目标，可能以漫画形式）
→ 教程（基础操作，鼠标点击交互）
→ 正式开始游戏（主要目标：把开发的游戏做完）
→ 结局判定（构思中）
→ [达成真结局] → 真结局endPV（总结剧情cg+参与人员名单，结尾显示打开制作的游戏）
→ 返回主界面，标题/背景UI改变
    （"100天独居生活" → "一个人的孤独之旅" / 100Days → AloneLife）
→ 正式进入内层游戏（玩家开发完成的游戏）
→ 通关内层游戏后出现双背景+双标题（可自由选择）
→ 正式进入二周目（可能没有新内容）
```

## 真结局后的内容

1. 播放endPV（总结剧情CG + 参与人员名单）
2. 结尾显示"玩家打开他制作的游戏"
3. 返回主界面，UI改变：
   - 标题改为"一个人的孤独之旅（AloneLife）"
   - 背景更换
4. 正式进入内层游戏（类兰斯10回合制RPG，TP系统）
5. 通关内层游戏后：
   - 双背景+双标题显示，可自由选择玩哪个
   - 进入二周目（可能没有新内容）
